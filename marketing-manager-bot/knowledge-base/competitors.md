# Competitors

Use this file to track competitors and alternatives. Do not attack competitors in customer-facing copy.

## Competitor Template

```yaml
competitor_id: ""
name: ""
website: ""
category: ""
target_market: ""
positioning: ""
strengths: []
weaknesses_internal_only: []
comparison_notes: ""
source_urls: []
last_checked_at: ""
```

## Usage Rules

- Competitor weaknesses are internal-only.
- Do not write negative competitor claims in outreach.
- Use competitor research to clarify positioning, not to attack.
