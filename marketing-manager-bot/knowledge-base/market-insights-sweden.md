# Market Insights Sweden

Use this file to store Swedish market research notes. Any claim that can change over time must include a source URL and checked date.

## Insight Template

```yaml
insight_id: ""
topic: ""
summary: ""
relevance_to_products: ""
source_url: ""
source_name: ""
checked_at: ""
confidence: "low|medium|high"
notes: ""
```

## Research Rules

- Prefer official Swedish sources, industry reports, company websites, and reputable publications.
- Separate facts from interpretation.
- Do not use outdated market claims without checking them again.
- Note if an insight is only an assumption.
