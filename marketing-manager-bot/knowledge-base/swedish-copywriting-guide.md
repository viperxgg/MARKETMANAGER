# Swedish Copywriting Guide

## Purpose

Write Swedish content that feels natural, professional, and credible for the Swedish market.

## B2B Style

- Use simple Swedish.
- Be specific without being pushy.
- Prefer practical value over hype.
- Use soft CTAs.
- Respect the customer's current business.

## Good CTA Examples

```text
Vill ni se hur flödet fungerar i praktiken?
```

```text
Om det känns relevant visar vi gärna ett kort exempel.
```

## Avoid

- "Vi garanterar fler kunder"
- "Ni förlorar pengar"
- "Ert system är dåligt"
- Fake urgency
- Excessive emojis or hype

## Platform Notes

- Facebook: slightly longer, trust-building, explanatory.
- Instagram: shorter, visual, stronger hook.
- LinkedIn: professional, founder/technical credibility, business outcome tone.
