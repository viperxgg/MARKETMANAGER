# Social Content Planning Workflow

## Goal

Create a balanced Facebook and Instagram content plan for a new page.

## Required Inputs

- Product positioning
- Audience notes
- Offer facts
- Approved visual direction
- Available proof, examples, or demo assets

## Steps

1. Social Media Strategist Agent defines the weekly theme.
2. Campaign Planning skill balances content across awareness, trust, education, examples, offer, and soft CTA.
3. Swedish Copywriter Agent writes Facebook drafts.
4. Platform Adapter Agent adapts each idea for Instagram.
5. Visual Creative Agent creates image concepts and English image prompts.
6. Compliance Review Agent checks claims, tone, and visual risk.
7. Human Approval Gate sends the plan to owner review.
8. Publisher & Scheduler Agent prepares schedule only after approval.

## New Page Content Mix

- Introductory posts
- Educational posts
- Problem-and-solution posts without attack language
- Trust-building posts
- Demo posts
- Soft invitation posts

## Output Fields
