# Workflows

These workflows define how the agency system should operate.

## Workflow Order

1. `product-onboarding.md`
2. `lead-generation.md`
3. `personalized-outreach.md`
4. `social-content-planning.md`
5. `approval-flow.md`
6. `analytics-feedback.md`

## Core Rule

Workflows can produce drafts and recommendations. They cannot send, publish, schedule, or contact real leads unless `approved_by_owner = true`.
