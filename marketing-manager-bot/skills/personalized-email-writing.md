# Skill: Personalized Email Writing

## Purpose

Write personalized Swedish outreach emails that are respectful, relevant, and approval-ready.

## Inputs

- Lead record
- Verified contact
- Website analysis
- Product offer
- CTA
- Sender identity placeholder

## Method

1. Start with a short, respectful greeting.
2. Mention one company-specific observation in a non-critical way.
3. Connect the product to a clear opportunity.
4. Keep the body concise.
5. Use a soft CTA such as asking if a short demo would be relevant.
6. Add opt-out text when appropriate.
7. Keep status as draft or owner review.

## Output Format

```yaml
email:
  subject_options_sv: []
  body_sv: ""
  personalization_used: ""
  cta_sv: ""
  opt_out_text_sv: ""
  status: "draft"
  approved_by_owner: false
```

## Draft Structure

```text
Hej [Företagsnamn],

[Respectful company-specific context.]

[Opportunity-based connection to the product.]

[Soft CTA.]

Vänliga hälsningar,
[Sender]

[Opt-out text if needed.]
```

## Quality Checklist

- Personalized to the company.
- Does not criticize the company.
- Does not overpromise.
- Clear business reason.
- Soft CTA.
- Ready for compliance review.
