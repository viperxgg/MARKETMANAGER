# Skill: Facebook Content Creation

## Purpose

Create Facebook posts for a new Swedish business page.

## Inputs

- Campaign theme
- Product facts
- Audience
- Content objective
- Approved claims

## Method

1. Choose one objective: awareness, education, trust, demo, example, or soft offer.
2. Write Swedish copy with slightly more context than Instagram.
3. Use short paragraphs.
4. End with a soft CTA when appropriate.
5. Avoid sounding like a direct ad on every post.
6. Mark as draft for review.

## Output Format

```yaml
facebook_post:
  objective: ""
  copy_sv: ""
  cta_sv: ""
  visual_brief: ""
  status: "draft"
  approved_by_owner: false
```

## Quality Checklist

- Useful for a new page with low trust.
- Clear enough without being long-winded.
- No unsupported claims.
- CTA is low-pressure.
- Suitable for Swedish audience expectations.
