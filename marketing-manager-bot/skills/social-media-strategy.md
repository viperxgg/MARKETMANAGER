# Skill: Social Media Strategy

## Purpose

Design a trust-building Facebook and Instagram strategy for a new page.

## Inputs

- Product positioning
- Audience
- Campaign goal
- Available visual assets
- Posting capacity

## Method

1. Define the weekly objective.
2. Choose a balanced mix of education, trust, demo, examples, and soft CTA.
3. Plan posts for Facebook and Instagram with platform-specific roles.
4. Keep the page from sounding like only direct sales.
5. Include image concept needs.
6. Send all posts to approval before publishing.

## Output Format

```yaml
social_strategy:
  week: ""
  objective: ""
  content_mix:
    awareness: 0
    education: 0
    trust: 0
    demo: 0
    offer: 0
  planned_posts: []
  approval_required: true
```

## Quality Checklist

- Suitable for a new page.
- Builds trust before selling.
- Uses platform-specific copy.
- Has visual direction for every post.
- Requires owner approval before publishing.
