# Skill: Website Opportunity Analysis

## Purpose

Analyze a lead's website to identify respectful opportunity angles for outreach.

## Inputs

- Lead website
- Product positioning
- Target customer journey
- Relevant pages to inspect

## Method

1. Review the homepage for clarity of message.
2. Review the main customer action path, such as booking, ordering, menu viewing, or contact.
3. Note visible friction without judgmental language.
4. Identify where the product could make the journey simpler or more modern.
5. Convert findings into outreach-safe wording.
6. Mark sensitive or negative details as internal-only.

## Output Format

```yaml
website_analysis:
  summary: ""
  customer_journey_notes: ""
  internal_opportunities: []
  outreach_safe_angle_sv: ""
  do_not_mention: []
  confidence: "low|medium|high"
```

## Outreach-Safe Wording Pattern

```text
Det finns en möjlighet att göra kundresan tydligare och mer smidig för era kunder.
```

## Avoid

- Calling the website weak, old, bad, or confusing.
- Claiming conversion loss without proof.
- Mentioning private or sensitive observations.
