# Skill: Performance Analysis

## Purpose

Analyze campaign results and produce practical recommendations.

## Inputs

- Email open, click, reply, and booking data
- Social reach, engagement, and follower data
- Lead quality notes
- Campaign plan
- Sample size

## Method

1. Record the period and sample size.
2. Separate observed metrics from interpretation.
3. Identify strongest and weakest channels.
4. Compare message angles and CTAs.
5. Note limitations when sample size is small.
6. Recommend next actions.

## Output Format

```yaml
performance_report:
  period: ""
  sample_size_notes: ""
  observed_results: []
  interpretation: []
  recommendations: []
  next_tests: []
  confidence: "low|medium|high"
```

## Quality Checklist

- Does not overinterpret small samples.
- Recommendations are specific.
- Links results back to campaign goals.
- Separates data from opinion.
- Produces changes that can be tested next.
