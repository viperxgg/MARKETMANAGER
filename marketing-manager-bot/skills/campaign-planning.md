# Skill: Campaign Planning

## Purpose

Plan a coherent weekly or monthly campaign from product positioning.

## Inputs

- Product positioning
- Target audience
- Campaign goal
- Channels
- Available proof
- Owner constraints

## Method

1. Define campaign goal and success metric.
2. Choose one primary audience.
3. Select 2-4 message angles.
4. Map content across awareness, trust, education, demo, offer, and CTA.
5. Define required assets.
6. Define approval points.
7. Keep launch volume small for early testing.

## Output Format

```yaml
campaign:
  campaign_id: ""
  goal: ""
  audience: ""
  channels: []
  message_angles: []
  weekly_plan: []
  required_assets: []
  approval_required: true
  status: "draft"
  approved_by_owner: false
```

## Quality Checklist

- Campaign has one clear goal.
- Message angles match product facts.
- Content is not only sales-focused.
- Approval points are clear.
- Metrics can be reviewed later.
