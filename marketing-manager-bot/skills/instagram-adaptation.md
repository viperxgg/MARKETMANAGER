# Skill: Instagram Adaptation

## Purpose

Adapt approved or draft content into Instagram-friendly Swedish captions.

## Inputs

- Base post idea
- Facebook draft or campaign note
- Visual concept
- Hashtag direction

## Method

1. Keep the core claim unchanged.
2. Make the caption shorter and more visual.
3. Use a first line that gives context quickly.
4. Add a soft CTA if useful.
5. Suggest relevant Swedish or industry hashtags.
6. Keep the caption readable without relying on hashtags.

## Output Format

```yaml
instagram_post:
  caption_sv: ""
  cta_sv: ""
  hashtags: []
  visual_note: ""
  status: "draft"
  approved_by_owner: false
```

## Quality Checklist

- Shorter than Facebook version.
- Fits the visual concept.
- Uses relevant hashtags only.
- Does not add new claims.
- Calm and professional.
